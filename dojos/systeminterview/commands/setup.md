# Setup Command

1. Create `workspace/` directory if it doesn't exist
2. Check that `curriculum/` and `curriculum/reference/` exist
3. Verify `bin/dojo.sh` is executable
4. Initialize progress: run `bin/dojo.sh init` (creates `.systeminterview/progress.json` with step=1)
5. Confirm setup is complete

No external dependencies needed for this dojo — it's a design interview, not a coding exercise. The candidate writes design documents, not code.
